# ProjectManagementExpo

**ProjectManagementExpo** es una exposición desarrollada para el curso de **Calidad de Software** del **quinto semestre de Ingeniería de Software**.  
Este proyecto tiene como objetivo demostrar las habilidades adquiridas en la gestión y desarrollo de proyectos de software, incluyendo la implementación de diversas interfaces y juegos.

## 🛠 Tecnologías utilizadas

El proyecto incorpora una variedad de herramientas y tecnologías, entre las que se incluyen:

- **HTML**: Lenguaje de marcado utilizado para estructurar el contenido de las páginas web.
- **CSS**: Hojas de estilo en cascada empleadas para diseñar y maquetar las interfaces de usuario.
- **JavaScript**: Lenguaje de programación utilizado para agregar interactividad y dinamismo a las páginas web.

📊 **Distribución del código en el proyecto:**
- HTML: **42.5%**
- CSS: **33.0%**
- JavaScript: **24.5%**

> *Nota: Las proporciones mencionadas se basan en la estructura del repositorio del proyecto.*

## 📁 Contenido del proyecto

El proyecto incluye:

- **Ejemplos de la aplicación** 🎨  
  Demostraciones de cómo funcionaría la aplicación final, destacando sus funcionalidades principales.
- **Interfaces de usuario** 📱  
  Diseños detallados de las distintas pantallas y componentes que conforman la aplicación.
- **Juegos interactivos** 🎮  
  Implementaciones de juegos que ejemplifican el uso de las tecnologías mencionadas y la aplicación de principios de calidad de software.
